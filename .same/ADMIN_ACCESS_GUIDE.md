## 📝 How to Edit Website Content

### Method 1: Content Blocks (Recommended)
1. Login to admin
2. Click "Content Blocks"
3. Find the content you want to edit
4. Click edit button
5. Change the text/content
6. Click Save

### Method 2: Direct Database
1. Each piece of content has a unique "key"
2. Example: `home_hero_title`
3. Edit the value to change what appears on website

---

## 🔧 Common Tasks

### Change Navigation Menu
1. Go to "Menu Manager"
2. Click "Add Menu Item" or edit existing
3. Set label, link, and visibility
4. Save changes

### Hide a Page Section
1. Go to "Content Blocks"
2. Find the section
3. Toggle visibility to "Hidden"
4. Content disappears from website

### Edit Button Text
1. Go to "Content Blocks"
2. Search for the button
3. Edit the value
4. Save - button updates instantly

## 💡 Pro Tips

 Use unique keys for content blocks (e.g., `about_hero_title`)
 Test visibility changes in new browser tab
 Content blocks can be reused across pages
 Menu items can be ordered by dragging
